# مقدمه

<div align="center">

![pvz logo](../../assets/logo/pvz.png)

</div>

پروژه درس برنامه‌نویسی پیشرفته نسخه‌ای از بازی Plants vs. Zombies 2 خواهد بود.

در این بازی، بازیکن در یک دنیای خیالی و آخرالزمانی قرار دارد که توسط زامبی‌ها تصاحب شده است. مأموریت اصلی حفاظت از آخرین خطوط دفاعی انسان‌ها با استفاده از انواع گیاهان است. هر گیاه دارای توانایی‌های منحصربه‌فرد و اثرات ویژه‌ای است که بازیکن باید بر اساس نوع تهدید و شرایط مرحله‌ی بازی، استراتژی مناسبی را انتخاب کند.
