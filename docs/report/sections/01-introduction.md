# قوانین و نکات قابل توجه {#rules}

- این سند نسخه‌ی کامل‌تر داک توضیحات بازی است و تمرکز آن روی شرح دقیق مکانیزم‌ها و انتخاب‌های اجرایی پروژه است.
- ترتیب کلی محتوا: منوها، روند اصلی بازی و سختی، گیاهان و زامبی‌ها، مراحل و باس‌ها، و سپس زیرسیستم‌ها.
- در هر بخش، فقط مواردی انتخاب شوند که هم ارزش طراحی داشته باشند و هم پیاده‌سازی آن‌ها در محدوده پروژه منطقی باشد.
- موارد بسیار پیچیده فقط در صورت توجیه فنی روشن وارد شوند.
