# مکانیزم‌های ثابت مراحل {#fixed-stage-mechanisms}

**مسئول: رهام**

- مکانیزم تولید آفتاب (نرخ، محدودیت، منابع تولید) دقیق تعریف شود.
- منطق تولید زامبی‌ها، موج‌بندی و تغییر سختی بین مراحل توضیح داده شود.
- تعداد خطوط هر مرحله و اثر آن روی طراحی استراتژی بازیکن مشخص گردد.
- روند کلی پیشروی بازی و بخش باقی‌مانده مرتبط با الگوریتم سختی تکمیل شود.
